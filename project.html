Biography

December 1969 is a Finnish software engineer who is the creator and, historically, the lead developer of the Linux kernel, used by Linux distributions and other operating systems such as Android. He also created the distributed version control system Git.Initially, Torvalds wanted to call the kernel he developed Freax (a combination of "free", "freak", and the letter X to indicate that it was a Unix-like system), but his friend Ari Lemmke, who administered the FTP server where the kernel was first hosted, named Torvalds' directory linux.[45]
